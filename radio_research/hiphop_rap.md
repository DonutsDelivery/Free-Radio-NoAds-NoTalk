## Notes

### Recommended Priority (Most Reliable)
1. **SomaFM Fluid** - Most reliable direct stream URLs, excellent quality, well-maintained
2. **181.FM Old School** - Established network with consistent streams
3. **PARTY VIBE RADIO** - High bitrate (320kbps), proven reliability

### Stream URL Types
- Direct URLs (ending in .mp3, .aac, /stream) work best with media players
- PLS/M3U playlist files contain redirect URLs
- Some stations only offer web players - use browser network inspector (F12 > Network > Media) to extract direct URLs
