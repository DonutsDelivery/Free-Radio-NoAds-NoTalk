## Notes on Finding Stream URLs

Many stations do not publish direct stream URLs openly. To find them:

1. **PLS/M3U files**: Download the station's playlist file and open in a text editor
2. **Browser Developer Tools**: Open Network tab, filter by Media, start the stream and look for .mp3/.aac URLs
3. **Stream Detector Extension**: Browser add-on that captures stream URLs
4. **Directory Sites**: internet-radio.com, rcast.net often list direct URLs
