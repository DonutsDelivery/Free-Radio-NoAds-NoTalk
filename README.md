## Features

- Stream ad-free internet radio stations
- Curated stations from RadCap, SomaFM, and Radio Paradise
- Save favorite stations
- Volume control with system integration
- Spectrum visualizer
- Works on any desktop environment (standalone app) or as a Plasma widget

## Plasma Widget

For KDE Plasma users, a dedicated widget is available that integrates with your panel.

See `freeradio/README.md` for widget installation instructions.
